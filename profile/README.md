### Installation Steps

1. Extract the trainer files.
2. Run TheMoundTrainer.exe as Administrator.
3. Launch the game and enter a save.
4. Press F12 to attach.
5. Open menu with INSERT.
6. Activate modules as needed.
7. Save custom presets.
8. Detach with F11 on exit.
